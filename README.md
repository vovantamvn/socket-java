# Socket Java

## Build and run server

> - cd src/com/company/server (Đứng ở thư mục server)
> - javac *.java
> - cd ../../..
> - java com.company.server.Server (Đứng tại thư mục src)

## Build client

> - cd src/com/company/client (Đứng ở thư mục client)
> - javac *.java
> - cd ../../..
> - java com.company.client.Client Client_Name (Đứng ở thư mục src, Client_Name là tham số truyền vào)
